# Cassandra

![Mou icon](../img/Cassandra.png)

版本：v3.4

Cassandra 是一套开源分布式 NoSQL 数据库系统。基于 Column 的结构化，拥有良好的伸展性。

查看后端服务、申请后端服务实例、绑定后端服务实例参见功能介绍《第四节 后端支持服务》章节。
