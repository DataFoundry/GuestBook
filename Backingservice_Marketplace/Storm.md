# Storm

![](img/Storm.png)

版本：v1.5.2

Storm 为分布式实时计算框架。可以方便地在一个计算机集群中编写与扩展复杂的实时计算。

### 使用 Storm 实例

- 使用 bind 返回的 host、port 连接 storm nimbus （zookeeper 信息提供在 uri 中）。
- 使用 bind 返回的 uri 中 storm-UI 访问管理 storm supervisors 以启动一定数量的 storm workers。
